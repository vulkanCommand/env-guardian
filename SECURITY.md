# Security Policy

Env Guardian is a security-focused CLI. Please report security issues responsibly.

## Reporting a Vulnerability

Email:

```text
gdkalyan2109@gmail.com
```

Include:

- affected version
- operating system
- reproduction steps
- expected behavior
- actual behavior
- whether any secret or sensitive data was exposed

Please do not include real secrets in reports. Use placeholders whenever possible.

## Supported Versions

The latest version on `main` is the supported development version until tagged releases are published.

## Public Issues

For non-sensitive bugs and feature requests, use:

```text
https://github.com/vulkanCommand/env-guardian/issues
```
